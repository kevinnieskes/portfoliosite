QAService

ASP.net Salesforce application

Made By: Kevin Nieskes, Brandon G And Steven F

GitHub: QAService

This is an ASP .NEt application designed to solve a problem we encountered in the class room. This program displays the users questions on the main page so that they can wait to be answered by an instructor in person. It also logs them in saleforce so that data can be gathered from them, such as tiem it takes to answer each question and what type of questions are being asked.
